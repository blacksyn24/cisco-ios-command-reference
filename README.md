# 📡 CCNA Command Hub

Guide de référence interactif des commandes réseau pour les étudiants CCNA.

## ✨ Fonctionnalités
- 200+ commandes Cisco IOS & MikroTik RouterOS
- Recherche en temps réel dans les commandes
- Bouton COPIER pour chaque commande
- Exemples concrets pour chaque commande
- 100% HTML/CSS/JS — aucune installation, aucun framework

## 📦 Contenu
- ⚡ Switch Cisco — Navigation, VLANs, STP, SSH, Sécurité
- ◈ Routeur Cisco — Interfaces, Routage, OSPF, NAT, ACL, DHCP
- 💻 PC / Hôte Windows — ipconfig, ping, tracert, arp, netstat
- 🔴 MikroTik RouterOS — IP, Routage, DHCP, NAT, Firewall, Wi-Fi

## 🚀 Utilisation
Aucune installation requise. Télécharge et ouvre simplement le fichier 
index.html dans ton navigateur.
