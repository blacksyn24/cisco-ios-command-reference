# cisco-ios-command-reference
📡 CCNA Command Hub — Guide de référence interactif pour les étudiants en réseau.  Plus de 200 commandes Cisco IOS et MikroTik RouterOS organisées par catégorie,  avec le rôle et un exemple concret pour chacune. Inclut Switch, Routeur, PC/Hôte  et MikroTik. Entièrement en HTML/CSS, aucune installation requise.
